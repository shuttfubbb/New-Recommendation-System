# NAML
Neural News Recommendation with Attentive Multi-View Learning

This code base on Chuhan Wu's paper that called [Neural News Recommendation with Attentive Multi-View Learning](https://arxiv.org/pdf/1907.05576)

Dataset used on this code from MIND dataset. For more information please visit [MIND dataset](https://msnews.github.io/)
